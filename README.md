# Bike Purchase Behavior Analysis Dashboard 
## Overview

This project presents an Excel dashboard analyzing a bike sales dataset to uncover patterns in customer purchase behavior.
The analysis focuses on how demographic and socioeconomic factors influence the likelihood of purchasing a bike.

The dashboard enables dynamic exploration of trends using filters, helping identify key drivers behind customer decisions.

## Dataset Description

The dataset includes customer-level information such as:

- Demographics: Age, Gender, Marital Status, Region
- Socioeconomic Factors: Income, Occupation, Education
- Lifestyle Indicators: Number of Children, Commute Distance
- Assets: Home Ownership, Car Ownership
- Target Variable: Bike Purchased (Yes/No)
  
## Objectives
- Analyze bike purchase patterns across different customer segments
- Identify key factors influencing purchase decisions
- Explore relationships between income, age, and bike ownership
- Understand how commute distance impacts buying behavior
- Enable interactive filtering for deeper segmentation analysis
